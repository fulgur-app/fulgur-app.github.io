fulgur::fulgur::utils

Module paths 

Source

Path utilities for Fulgur configuration and data files.

This module provides functions for determining platform-specific configuration directory paths. The configuration directory is:

  • Windows: %APPDATA%\Fulgur
  • macOS/Linux: ~/.fulgur
